Yoga Pose Prediction and Realtime Feedback System

Python • Tensorflow • Image Processing • Scikit-learn • Custom Geometry Algorithms • Flask

The yoga pose prediction system begins with image preprocessing where input frames are resized and augmented to improve model robustness. MoveNet, a lightweight pose estimation model, then detects 17 key body points with high precision. These key points feed into a Random Forest classifier that identifies specific yoga poses by analyzing spatial relationships between joints. Finally, the system calculates joint angles using vector geometry, compares them to ideal pose benchmarks, and generates real-time feedback on alignment deviations through visual overlays and corrective suggestions.

Early-Stage Diabetes Risk Assessment Using Balanced Classification Models

Python • Scikit-learn • Imbalanced-learn • Data Visualization • Feature Engineering

The project begins with comprehensive data exploration and visualization to understand diabetes prevalence patterns and feature correlations. After addressing class imbalance using SMOTE, three classification models (Logistic Regression, Linear SVM, and Kernel SVM) are trained and evaluated using recall, precision, and ROC-AUC metrics. Feature importance analysis identifies key risk factors like general health, blood pressure, and BMI. The workflow culminates in an interactive diabetes risk prediction tool that takes user inputs, standardizes them, and provides risk assessment using the best-performing kernel SVM model.

House Price Prediction System

Python • Data Preprocessing • Data Visualization • Scikit-learn • Flask

Developed a House Price Prediction Model during an internship at Enlighten Infosystems, leveraging Python, Flask, and MySQL to build a scalable ML solution. The system processes real estate data (square footage, location, etc.) using Linear Regression, Random Forest, and LightGBM, achieving optimized accuracy through hyperparameter tuning. A Flask-based web app enables users to input property details and receive instant price estimates, supported by a secure MySQL backend. This project enhanced my skills in ML deployment, EDA, and full-stack integration, delivering a practical tool for real estate valuation.
